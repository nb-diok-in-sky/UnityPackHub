# UnityPackHub 需求清单

## 项目概述

Unity `.unitypackage` 资产包桌面管理工具，橱窗式卡片 UI，Apple 设计语言，支持中英双语。

技术栈：Tauri 2 + Vue 3 + Quasar + TypeScript + Pinia + Dexie (IndexedDB)

---

## 功能清单

### 1. 橱窗卡片展示 ✅
- [x] 网格布局，响应式排列
- [x] 每张卡片 = 封面图 + 名称 + 备注摘要 + 标签 + 文件大小
- [x] 三种卡片尺寸切换（小/中/大）
- [x] 悬浮提升动效，选中高亮边框

### 2. 自定义封面图 ✅
- [x] 详情面板拖拽图片设置封面
- [x] 点击上传图片设置封面
- [x] 移除封面按钮
- [x] 无封面时显示首字母占位符（渐变背景）
- [x] 自动从 Unity Asset Store 搜索同名资产抓取封面图
- [x] 详情面板 "自动获取封面" 按钮

### 3. 备注/描述 ✅
- [x] 详情面板双击编辑备注
- [x] Ctrl+Enter 或失焦保存
- [x] 卡片上显示备注摘要（最多两行）

### 4. 标签分类 ✅
- [x] 侧边栏新建标签（名称 + 颜色选择）
- [x] 右键编辑/删除标签
- [x] 详情面板给资产添加/移除标签
- [x] 8 种 Apple 风格预设颜色
- [x] 侧边栏点击标签筛选资产

### 5. 搜索过滤 ✅
- [x] 顶栏搜索框，实时过滤
- [x] 按名称、标签名、备注关键字搜索
- [x] 清除搜索按钮

### 6. 资产包目录扫描 ✅
- [x] 设置面板添加扫描目录
- [x] 递归扫描 `.unitypackage` 文件
- [x] 自动同步数据库（新增/移除）
- [x] 扫描加载动画
- [x] 启动时自动扫描已配置目录

### 7. 一键导入 Unity ✅
- [x] 详情面板 "导入到 Unity" 按钮（调用系统关联打开 .unitypackage）
- [x] 卡片双击直接导入
- [x] 右键菜单 "导入到 Unity"
- [x] 右键菜单 "打开文件位置"
- [x] 详情面板 "打开文件位置" 按钮

### 8. 资产包预览 ✅
- [x] Rust 端解析 `.unitypackage`（tar.gz），只读 `pathname` 条目，不解压 asset
- [x] 详情面板展开/折叠查看包内文件列表
- [x] 显示文件数量统计
- [x] 从包内提取 `preview.png` 作为封面图（替代 Asset Store 抓取）

### 9. 收藏/置顶 ✅
- [x] 卡片悬浮显示收藏按钮（星标）
- [x] 详情面板收藏按钮
- [x] 侧边栏 "收藏" 筛选
- [x] 收藏的资产自动排在前面

### 10. 批量操作 ✅
- [x] Ctrl+Click 多选
- [x] 全选 / 取消选择
- [x] 批量打标签 / 移除标签
- [x] 批量收藏 / 取消收藏
- [x] 批量删除（确认对话框）
- [x] 撤销操作（Ctrl+Z）

### 11. 多目录管理 ✅
- [x] 设置面板显示所有扫描目录
- [x] 启用/禁用单个目录
- [x] 移除目录
- [x] 设置持久化到 JSON 文件
